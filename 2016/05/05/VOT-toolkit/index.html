VOT toolkit（2015+2016）和TraX安装配置

发表于 2016-05-05 更新于 2016-05-18 Disqus：

vot-toolkit

正好前几天VOT放出来了今年的toolkit和相关文档，顺便更新一下。

2015部分

先下载

set up workspace

下载后在toolkit目录里运行MATLAB，或者把toolkit的目录addpath进去。然后做

给自己的tracker设置一个name，不出意外的话稍等几秒就创建完成。这时候目录里会多出来一个configuration.m和一个tracker_.m，这样workspace就设置完成了。
顺带一提之前一直都不知道MATLAB还可以直接在命令行下用_( :3 」 )_

integrating tracker

进入workspace目录，编辑刚刚生成的tracker_.m

  1. 把第一行error注释掉
  2. 把tracker_label后面的方括号里填上tracker的name，用引号打起来
  3. tracker_command里填上可执行的绝对路径，如果需要的话加上参数
  4. 如果需要在tracker_linkpath里加上要链接的库，也是绝对路径
    做完之后去run_experiments.m里check一下tracker_load后面tracker的name是不是对的，一般都不会有问题。

接下来的步骤最好参考/tracker/examples中来，这里只做了Matlab trackers

performing evaluation

2016部分

clone下来后发现比之前增加了一些东西：

  1. Select one of available experiment stacks的时候多了vot2015和vottir2015的选项
  2. 输入tracker标识符后可以选择tracker的语言，共有四种可以选择
    1. Matlab
    2. Python
    3. C/C++
    4. 其他
  3. 在tracker_.m里，tracker_command多了一个script name的选项，另外上一步选择的类型也会显示在这里
